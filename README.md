# Claude Design System

This repository archives a Chinese write-up of the Claude Design system prompt and a reader-friendly English translation of the `# 内容` section.

## Files

- `揭秘 Claude Design 系统提示词（中文版）.md`: the original Chinese note, including a summary and the full translated content.
- `claude-design-system-prompt-en.md`: an English version of the `# 内容` section. Code identifiers, URLs, and tool names are kept as-is where it improves accuracy.
